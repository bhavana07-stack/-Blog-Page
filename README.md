# Insight Blog

## Project Overview

Insight Blog is a static blog webpage developed using HTML, CSS, and JavaScript.

The website displays blog articles with:

- Title
- Author Name
- Publication Date
- Summary
- Additional article content

Users can expand or collapse blog content using the Read More button.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts

---

## Features

### Blog Posts
Displays three sample blog articles.

### Read More Functionality
Allows users to expand and collapse blog content.

### Responsive Design
Works on desktop, tablet, and mobile devices.

### Modern Styling
Includes:
- Gradient backgrounds
- Custom Google Fonts
- Hover animations
- Responsive card layout

## How to Run

1. Download the project files.
2. Open the project folder.
3. Open index.html in any web browser.

OR

1. Open the folder in VS Code.
2. Install Live Server.
3. Right-click index.html.
4. Select "Open with Live Server".
